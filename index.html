Risk Management App

This web application was developed for an Oil consulting firm. They were using project management tools like Oracle Primavera in conjunction with Microsoft Excel but was looking for a solution to combine those functionalities also adding new ones one of which is simulation of all possible paths(consisting of tasks,risks,solutions) alongside corresponsing cost and time. You can see this in action in demo (Analytics/simulation). Because of all complicated data structures (drag/drop tree-home page) and all those charts I had to use third party components (Kendo React-Telerik). Our tech stack is SQL Server + ASP.net for backend and React for frontend.
